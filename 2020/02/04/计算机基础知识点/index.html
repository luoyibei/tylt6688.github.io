计算机起源与发展历程

置顶 | 发表于 2020-02-04 分类于 计算机基础知识 阅读次数：

计算机基础知识大全

信息与数据

信息论的创始人：美国数学家香农，信息的功能是消除事物的不确定性，把不确定性变成确定性。

信息可以用不同形式的数据表示
  • 信息是对数据进行加工后的结果，信息的符号化就是数据
  • 数据是信息的载体，是信息的具体表现形式
  • 信息是数据的逻辑意义，数据是信息的物理表示

计算机文化

  • 出现在20世纪80年代初
  • 在瑞士洛桑召开的第三次世界计算机教育大会上

文化的核心：观念和价值
基本属性：广泛性、传递性、教育性、深刻性

计算机应用

主机：中央处理器CPU + 内存储器

  1. 科学计算
  2. 信息管理
  3. 过程控制
  4. 计算机辅助系统
    • 计算机辅助设计（CAD）
    • 计算机辅助制造（CAM）
    • 计算机辅助测试（CAT）
    • 计算机辅助教学（CAI）
    • 计算机辅助教育（CBE）
    • 计算机管理教学（CMI）
    • 计算机集成制造系统（CIMS）
  5. 人工智能
  6. 计算机网络与通信
计算机的发展趋势：

巨型化，微型化，智能化

计算机性能指标：

​ 字长，运算速度，主频（时钟频率），内核数，内存容量

运算速度的单位 MIPS(Million Instruction Per Second 每秒$10^6$条指令 ) BIPS(Billion Instruction Per Second 每秒$10^9$条指令 )

计算机存储的基本单位是字节

一个字节由八位二进制位组成

科学计算时溢出，数值太大，CPU无法承受，超出机器所能表示的范围

计算机软件由程序＋文档组成

字是CPU一次 存取加工和传递的数据，这一次传递的数据的二进制位数也就称为字长，与CPU的型号有关

字节是衡量计算机性能的重要指标，字长越长，速度越快，精度越高

多国语言支持的Unicode字符集，是一种16位字符编码标准（一个字符用十六位二进制数表示两个字节）包含ASCII字符集（7位二进制表示，最多可以表示128个字符）,

ASCII码在计算机中表示为一个字节，最高位是0

同一字母的ASCII码小写比大写大32，16进制的话就是20H，也就是小写字母的顺序在大写字母之后

显示器分类：①CRT显示器（阴极射线管显示器）大头 ②LCD显示器（液晶显示器） ③等离子显示器
分辨率的单位为像素
24位及以上称为真彩显示器

CPU与其它部件传输的数据就是通过总线
总线分为数据总线，控制总线，地址总线（都是串行总线）

并行总线大部分为有针脚的总线，例如VGA，PS/2

主板的芯片组分为南北桥

CMOS芯片（电池） 储存时间，日期，磁盘参数与计算机配置信息

磁盘驱动器、磁带机属于输入输出设备

磁盘属于存储设备

编译程序产生目标程序，解释程序不产生目标程序

CD-ROM和DVD都是利用激光技术存储

汉字输入方法：拼音码，字形码，音形码，流水码

​

多媒体计算机系统

特点：①多样性②实时性③交互性④集成性

多媒体技术的核心技术，数据压缩与编码技术

流媒体数据的特点：① 连续性 ② 实时性 ③ 时序性

多媒体数据在计算机中最终以二进制形式存在

静态文件格式：BMP，GIF，JPEG；GIF比较特殊，也可以支持动画，MPEG属于动态图像文件格式

计算机操作系统

快捷键：

Ctrl + 鼠标拖动，复制文件

Alt + 鼠标拖动，创建相应的快捷方式

激活应用程序的文件菜单：Alt + F

Windows有两个通配符：？和 *

  • ？（问号）表示与一个任意字符相匹配

  • *（星号）表示零个或多个任意字符匹配

Windows7不区分文件名的大小写

回收站是硬盘上的一块特殊区域，只能存放相应硬盘的删除文件，像软盘、U盘不可以。

其实每个盘符下面各有一个回收站，桌面上的回收站是所有回收站的总和，回收站可以改变其大小，但是不能被删除。

剪贴板是内存上的一块区域

画图程序不是多文档应用程序

Windows文件组织结构是一种树状结构

对话框分为模式对话框与非模式对话框

主程序窗口被禁止的对话框称为模式对话框

窗口与对话框的区别是窗口有菜单栏对话框没有

非活动窗口不可以接受用户的键盘与鼠标输入等操作，只有活动窗口可以

计算机网络

功能：

①数据通信 ②资源共享 （最突出的特征） ③分布式处理 ④提高系统的可靠性

ping命令测试两台计算机是否连通

tracert命令显示经过的路径上各个路由器信息

Internet的前身是ARPANet

Internet服务提供商（Internet Service Provider）简称ISP

Internet下载文件是FTP功能

一级域名 ：com 商业组织；edu 教育组织；gov 政府部门；mil 军事机构；net 网络服务器；org 非营利性组织

网卡工作在物理层、数据链路层
功能：① 提供全球唯一的MAC地址，网卡的物理地址 ② 接受网络数据，转换为本机可识别的格式 ③ 将本机要传输的数据按照网络设备的格式上传到网络

中继器、集线器工作在物理层

网桥工作在数据链路层

交换机工作在数据链路层，可以取代集线器和网桥

路由器工作在网际层，路径选择，数据包的传送，网络设备互联的核心，用于实现不同类型网络的互联

中继器连接相同网络

网关工作在传输层，称为协议转换器，连接不同网络

LAN和WAN主要区别是通信距离与传输速率

双绞线的最大传输距离是100m，如果要加大传输距离，需要在两段双绞线之间安装中继器，最多可以安装四个中继器，也就是传输距离加中继器后可为500米。

TCP/IP四个网络层次：应用层，传输层，网际层，网络层

IP地址：

A类 ： 1126 127作为回路、回访、以及诊断功能地址
B类 ：128191
C类 ：192~223
D类 ：用于多点广播
E类 ：保留未用

由于C类前24位标识网络号，所以只有余下8位表示主机号，可表示2^8，256个，根据Internet网络管理规定，主机号不能全为0或1，所以只能表示254个C类地址。

子网掩码：判断任意两台计算机IP地址是否属于同一子网，网络标准位全为“1”，主机标准位全为“0”

URL 全球统一资源定位器，用于标识全球某个网络资源的唯一标志，是信息资源在网上业务类型和如何访问的统一的描述方法
格式为：协议名://主机域名或IP/路径名/文件名（https://www.tylt.xyz/index.jpg）

四大主干网络：
  1. （ChinaNet）中国公用计算机互联网
  2. （CerNet）中国教育和科研计算机网
  3. （CSTNet）中国科技信息网
  4. （ChinaGBN）中国金桥信息网

计算机病毒

可执行性，破坏性，传染性，潜伏性，针对性，衍生性，抗反病毒软件性

类型：引导区病毒，文件型病毒，混合型病毒，宏病毒（worm蠕虫病毒）

trojet，木马，人为设计的有破坏性的一组程序，具有隐蔽性

木马不是病毒，木马程序不依附于任何载体而独立存在，病毒必须依附其它载体存在并具有感染性

木马主要是窃取密码，包括两部分组成，控制器端和服务器端

Word

快捷键：

打开文件：Ctrl+O或Ctrl+F12

保存文件：Ctrl+S或Shift+F12

复制：按住Ctrl键拖动选定的文本或Ctrl+C

查找：Ctrl+F

替换：Ctrl+H

定位：Ctrl+G

分页：Ctrl+Enter

软回车：只换行不换段，Shift+Enter

选择矩形区域：Alt + 鼠标拖动

Word的主要功能：1.文字编辑和格式化

​ 2.图形图片图像的处理

​ 3.表格处理

​ 4.版本设计与打印

word表格中的数据不能排序，可以计算

对表格设置表格线的宽度选择开始中的边框和底纹

Word的五种视图：

页面视图、阅读版式视图、Web版式视图、大纲视图、草稿（普通视图）

Word的即点即输功能，只在页面视图和Web版式视图才能使用

页面视图和草稿视图可以显示水平标尺

草稿视图是适合文本编辑和录入的视图

页面视图可以看到分栏效果
阅读版式视图可以显示以图书的分栏样式

页面视图只能看到分页效果，草稿视图可以看到分页符

页眉和页脚只能在页面视图显示

图文混排是在页面视图方式下实现的

大纲视图用于显示文档标题的层级结构、长文档的浏览和设置

选定栏位于页面编辑区的左侧空白区域

通配符主要有 ？（一个字符）和 *（多个字符） ，需在英文状态下输入

剪切板最多可保存24块剪切或复制的内容

页眉页脚必须在不同窗口编辑

拼写和语法检查：

  1. 红色波浪线表示拼写错误或字库中无该字

  2. 绿色波浪线表示语法错误

波浪线属于不可打印字符，不影响打印效果

首行缩进：段落中的第一行缩进一定值，只针对第一行有效

悬挂缩进：段落中除了第一行外，其余所有行缩进一定值

段落是指两个回车键之间的全部字符

按回车键产生一个段落标记符，位置在段落的尾部

双击格式刷能够使用多次，单击格式刷只能使用一次。
取消使用格式刷：①按Esc键 ②点击格式刷按钮

Word具有自动分页功能，页满时会自动换页并插入一个软分页符（不可删除），除此外，还有硬分页符（人工分页符，可删除）

​ ① 布局选项卡
​ ② 分隔符
​ ③ 点击分页符按钮

word表格最多是63列、32767行

在表格中全选后按Delete键删除的是内容

设置表格的分页标题显示：点击布局选项卡中的重复标题行

图表的插入与编辑是使用的OLE技术（链接与嵌入式技术）

插入图片，编辑图片
浮动式对象：可以放置到页面的任意位置，允许与其它对象组合，还可以与正文实现多种形式的环绕
嵌入式对象：只能放置到文档插入点的位置，不能与其它对象组合，可以与正文一起排版，不能实现环绕

域：隐藏在文档中的一组特殊代码组成的命令

选择矩形文本区，使用鼠标和 Alt 键

Excel

快捷键：

在多个单元格输入相同信息：Ctrl+Enter

在当前工作表之前新插入一个工作表：Shift+F11

将选定的单元格扩展到最后一个使用的单元格：Ctrl+Shift+End

下一列：Tab

下一行：Enter

工作簿（book）.xlsx 一个工作簿由255个工作表（sheet）组成，初始创建时会有三个工作表

一个工作表最多有1048576行（从1开始），16384列（从A开始）

工作簿中至少存在1个工作表，并且是可视，不能全部隐藏

文本类型：

字母、汉字、数字、空格、其它字符，默认左对齐

数字作为文本，在单元格输入英文半角单引号’后输入数字，字长的影响，计算机最多精确到15位，超过后都以0显示，身份证号，邮政编码，公式函数，以0开头的数据（数值型文本）

数字类型：

默认右对齐

, （逗号）是千位符，每三位加一个

输入分数时先输入0和空格，否则会被认为日期型

数字长度超过11位就变为科学计数法

日期和时间数据类型：

默认右对齐，24小时制。输入十二小时制输入后加空格加AM或PM，表示上午或下午
① 输入当天日期：Ctrl + ;
② 输入当前时间：Ctrl + Shift + ;
③ 输入当前日期加时间：先 ① 输入空格 再Ctrl + Shift + :

公式输入时必须以 = 开头

单元格的引用：

​ 相对引用：默认方式
​ 绝对引用：$加在每一个行号和列号 “=$A$1”
​ 混合引用：只加一个$表示行绝对引用或列绝对引用

相对引用、绝对引用、混合引用相互转化可以使用F4

三维地址引用：格式 [工作簿]工作表名!单元格引用
例如[Book]Sheet!A3（表示Book工作簿的工作表的第三行第一列）

单元格引用运算符：

​ ：（冒号）区域表示运算符
​ （空格）区域交集运算符
​ ，（逗号）区域并集运算符

行高和列宽的调整：拖动鼠标调整；双击行或列的分割线可自动调整

选择性粘贴可以实现数据部分特性的复制

数据清单：类似数据库的特点，具有二维表性质的，行表示记录，列表示字段

进行分类汇总前必须要先根据分类的字段进行任意的排序

RANK(number,ref,order)number为排位的数字，ref是单元格区域，必须为绝对引用，order指明排序方式，0或缺省为降序，不为0则进行升序，对重复数排位相同，但会影响后续的数排序

图表是单元格数据以图显示，图表是对单元格数据的绝对引用，改变时都会进行相应的改变
一种是嵌入式图表，在同一张工作表中，另一种是独立图表

函数中的参数可以是常量，变量，单元格，区域，逻辑位，引用，错误值或其他函数

PPT

快捷键：

第一张幻灯片开始放映：F5或幻灯片放映选项卡中的从头开始

当前幻灯片开始放映：Shift+F5或从当前幻灯片开始放映

插入一个新幻灯片：Ctrl + M

.pptx是文件类型

.potx是模板类型

保存为幻灯片放映的文件：.ppsx 可直接进行播放

打包可以在无ppt环境下播放也是打包成 .pptx

视图：普通视图，幻灯片浏览视图，幻灯片放映视图

启动时不会自动创建新文档

演示文稿中的每一个单页被称为幻灯片，是演示文稿的核心，其中每个对象所在的位置被称为版式，可以改变幻灯片的布局。

模板、配色方案和母版是控制眼视文稿外观的重要手段

母版：幻灯片母版、讲义母版、备注母版

超链接技术以及动作设置可以制作更好的交互功能，链接目标不能是某张幻灯片的某个对象

占位符不可被插入

插入的超链接中不能是幻灯片中的某个对象

排练计时用于自动放映

Access

扩展名 .accdb

Access的六种数据库对象：表，查询，窗体，报表，宏，模块

选择：关系中满足条件的元组

连接：两个关系的笛卡尔积中选取属性满足一定条件的元组

投影：从关系中挑出若干个属性列组成新的关系

五种视图：
  • 设计视图
  • 数据表视图
  • SQL视图
  • 数据透视表视图
  • 数据透视图视图

Access管理的关系是若干个二维表格

二维表形式显示的是数据表视图

数据库管理系统：DBMS

数据库管理系统的功能：① 数据定义（DDL）② 数据存取 ③ 数据库运行管理（DML）④ 数据库的建立和维护 ⑤ 数据库的传输

数据库控制语言（DCL）

字段命名规则：

​ ① 长度最多为64个字符
​ ② 可以包含字母、数字、空格以及特殊的字符（除 . ! ‘ [ ] 外 ）
​ ③ 不能以空格开头
​ ④ 不能包含控制字符（从0~31的ASCII值）

窗体是数据库与用户交互的最好界面

报表是用于打印的

数值和文本可用于建立索引
